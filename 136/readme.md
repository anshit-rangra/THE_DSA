This is the solution of leetcode question 136 which is based on bit manupulation 
the question relies on XOR operation ( ^ ) . because this operator has unique property. when a and b are same so they cancel out each other in this operator. 

let's take example = [ 4, 1, 2, 1, 2 ]

0 ^ 4 = 4
4 ^ 1 = 5
5 ^ 2 = 7
7 ^ 1 = 6
6 ^ 2 = 4

so at last the answer is 4 that is a single number
